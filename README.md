# 🛒 E-commerce Automation using Selenium

## 📌 Project Overview
This project automates an end-to-end e-commerce workflow using **Selenium WebDriver (Python)**.  
It simulates a real user performing login, product selection, cart validation, checkout, and order completion.

The framework is built using the **Page Object Model (POM)** design pattern for better scalability and maintainability.

---

## 🚀 Features Automated

- ✅ User Login
- ✅ Product Sorting (Low to High)
- ✅ Add Product to Cart
- ✅ Cart Validation
- ✅ Checkout Flow
- ✅ Order Confirmation
- ✅ Screenshot Capture on Success
